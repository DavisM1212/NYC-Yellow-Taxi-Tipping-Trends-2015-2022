# NYC Yellow Taxi Tipping Trends (2015-2022)
A simple HTML website creating a heatmap showing average tipping behavior across NYC zones and boroughs.
Data was cleaned and aggregated from the publicly available TLC trip data parquets located at https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.page.

Heatmap has simple panning, tooltips, year toggles, and metric toggles.
